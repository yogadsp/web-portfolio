Laporin Aja (Layanan Pengaduan Official ITERA)

Laporin Aja is a web that contains response services regarding information on loss and damage reports to facilities on the ITERA campus. This website contains a feature that can report ITERA activists' responses about the facilities and report the discovery or loss of items. Another feature of this website is the presence of Live Chat which can be used to report missing items or damage to facilities which will be immediately responded by the admin. Furthermore, the admin will respond by immediately following up the report by giving an announcement on the website or reporting to the authorities at ITERA.
