Automatic Trash Bin Prototype

Prototype automatic trash bin was placed in each house. Waste that can be sorted consists of vegetables, fruits, paper, cardboard, plastics and metals. Sorting is assisted by a cloud service, namely Amazon Image Rekognition, which is able to recognize various objects based on images without prior training. In addition to sorting waste, the prototype can also detect whether the trash can is full or not. The Android application displays trash bin info consisting of its location and condition.

Tools :
  • Mobile : Android Studio
  • Web Server : ExpressJS, Amazon Image Rekognition, Azure Virtual Machine, JSON, MQTT
  • Hardware Compiler : Arduino IDE
